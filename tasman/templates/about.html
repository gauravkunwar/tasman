About this Project.

This project illustrates the objectives of developing an application that exhibits the variation in conductivity, depth and temperature across the Tasman Sea. The scope of this project ranges from the research purpose to the evaluation of variations underwater.

The students of Federation University undertaking Master Of Technology are supposed to undertake this project for ITECH 7415 (Masters Project) course. The project took off forming a team consisting four members and working under a business client and a project supervisor.

This project is solely related to the development of an Information System where underwater communication techniques can be observed. The interactive applet with database is integrated in the website, where the associated images and graphs are modelled in graphs.

Features for admin include '.csv' file upload for whole database update or singular point of data modification.
